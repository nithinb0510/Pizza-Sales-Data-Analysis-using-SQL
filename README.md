# 🍕 Pizza Sales Data Analysis using SQL

## 📌 Project Overview

This project focuses on analyzing pizza sales data using SQL to extract meaningful business insights. The goal is to understand customer behavior, sales trends, and product performance.

## 🗂 Database Structure

The project includes the following tables:

* **orders** – Stores order date and time
* **order_details** – Contains order-wise pizza details
* **pizzas & pizza_types** – Stores pizza information like category, size, and price

## 🔍 Key Analysis Performed

### Basic Analysis

* Total number of orders placed
* Total revenue generated
* Highest-priced pizza
* Most common pizza size
* Top 5 most ordered pizzas

### Intermediate Analysis

* Category-wise pizza demand
* Hourly order distribution
* Average pizzas ordered per day
* Top pizzas based on revenue

### Advanced Analysis

* Revenue contribution by category
* Cumulative revenue over time
* Top pizzas by revenue within each category

## 🛠 Skills & Concepts Used

* SQL Joins (INNER JOIN)
* Aggregations (SUM, AVG, COUNT)
* GROUP BY & ORDER BY
* Window Functions (RANK, OVER)
* Subqueries

## 📈 Key Insights

* Identified best-selling pizza categories
* Analyzed peak ordering hours
* Found top revenue-generating products
* Tracked business growth over time

## 🚀 Conclusion

This project demonstrates how SQL can be used to perform real-world data analysis and generate actionable business insights.
